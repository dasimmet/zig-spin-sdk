Hello World! from Zig Wasm

This is an Example on building an application on Fermyon Spin/Cloud using zig. Zig Programs are compiled to wasm32-wasi and assigned routes, then executed using WAGI (WebAssembly CGI). This page is served by the main route using zig's @embedFile builtin.

Examples:

  • JSON Example - Dumps out the Environment using tres
  • Key Value Store (not supported on cloud)
